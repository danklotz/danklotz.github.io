bp-exploration

Notes on using backpropagation as a tool for model exploration and interpretation.

Dstill.pub: Feature-Visualization is an amazing article that nicele exhibits how backprop can be used to visualize features (edges, textures, patterns, objects, etc.) of specific neural network parts/layers. Mordvintsev did quite some work in this direction. All looks great.

Looking Inside The Blackbox. This is blog-post/tutorial about the usage of backprop do obtain some qualitative understanding of neural-networks. Not to brad, but I had a similar idea in the context of “data-assimilation for Deep Learning based rainfall-runoff modeling”. As in the blogpost, the idea was to backprop with regard to the inputs (instead of the parameters), but with the goal to get (a) a better forecast and (b) obtain a qualitative understand about what the networks thinks about the world.
