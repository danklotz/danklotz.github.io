Feedback with tribbles

The ordering and display of tables in Rmarkdown (aka data.frames or tibbles) has become so good. I actually prefer it over using print right now. Especially, because I can format in in an intuitive way by using the tribble() function. Here is an example. Let us say I want to summarize the basic set up for an optimization. Some dummy variables might look like this:

library(tibble)

number_iterations <- 1000
stoping_criterion <- -1
stepsize <- 0.5
sensitivity <- 0.2

To give a (nicely formatted) summary in the past I would have done something like this:

library(pasta)

print("number of iterations:" %&&% number_iterations)
## [1] "number of iterations: 1000"
print("stoping criterion:" %&&% stoping_criterion)
## [1] "stoping criterion: -1"
print("step size:" %&&% stepsize)
## [1] "step size: 0.5"
print("sensitivity:" %&&% sensitivity)
## [1] "sensitivity: 0.2"

Now, the tribble functions allows me to order the feedback in a way that is nicely formatted in code and in display:

info <- tribble(
  ~criterion,             ~value,
  "number of iterations", number_iterations,
  "stoping criterion",    stoping_criterion,
  "step size",            stepsize,
  "sensitivity",          sensitivity
)

info
## # A tibble: 4 x 2
##              criterion  value
##                  <chr>  <dbl>
## 1 number of iterations 1000.0
## 2    stoping criterion   -1.0
## 3            step size    0.5
## 4          sensitivity    0.2
